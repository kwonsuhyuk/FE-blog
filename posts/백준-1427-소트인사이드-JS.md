### 문제 아이디어
입력받은 숫자를 배열로 변경해서 내림차순으로 정렬후에 다시 합쳐주어 풀어야 겠다고 생각했다.

### 코드
```js
let fs = require('fs')
let input = fs.readFileSync('/dev/stdin').toString().split('\n')

let n = input[0].split("").map(Number).sort((a,b)=>b-a)
let ans = n.join("")
console.log(ans)
```

### 코드 해석

`let n = input[0].split("").map(Number).sort((a,b)=>b-a)`

- 입력받은 숫자를 `split("")` 을 통해서 배열로 만들어준다.

---

`.sort((a,b)=>b-a)`

- comparefunction 으로 내림차순으로 정렬해준다. 

---

`let ans = n.join("")`

- 각 배열을 다시 문자열로 합쳐준다.
