## HTTP method
- 생성 : POST
- 조회 : GET
- 수정 : PUT(덮어쓰기) /PATCH (일부 변경)  
- 삭제 : DELETE

## Node.js 
- 특징 : 싱글 스레드, 이벤트 기반 , 논블로킹 I/O

## 모듈 (라이브러리)
- 미리 만들어둔 코드 덩어리

## 프레임워크
- 필요해 보이는 라이브러리(모듈)들을 미리 틀 안에 다 넣어둔것

## 콜백 함수
- 함수의 매개변수로 변수 또는 값을 전달하는 것이 아니라. 함수를 전달하는것.
