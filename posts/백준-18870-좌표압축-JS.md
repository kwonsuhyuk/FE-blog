### 문제 아이디어

좌표를 정렬해서 서로 다른 좌표의 개수를 구하기에 
`let newArr = [...new Set(arr)]` 를 이용해서 중복을 제거후에 정렬해서 각 원소의 인덱스를 출력해주면 된다고 생각하였다.

### 코드
```js
let fs = require('fs')
let input = fs.readFileSync('/dev/stdin').toString().split('\n')

let n = Number(input[0])
let arr = input[1].split(" ").map(Number)

// 서로 다른 좌표 -> 중복제거
let newArr = [...new Set(arr)]

newArr.sort((a,b) => a-b)

let map = new Map()
for (let i=0; i<newArr.length ; i++) {
  map.set(newArr[i],i)
}

let ans = ''
for (let x of arr) {
  ans += map.get(x) + " "
}
console.log(ans)

```

### 코드 해석
```js
newArr.sort((a,b) => a-b)

let map = new Map()
for (let i=0; i<newArr.length ; i++) {
  map.set(newArr[i],i)
}
```
- Map 을 사용해서 각 원소와 정렬한 배열의 인덱스를 키,벨류 값으로 넣어주었다. 
---

```js
let ans = ''
for (let x of arr) {
  ans += map.get(x) + " "
}
```
- 그 후에 원래 배열을 돌면서 최종 답에 꺼내서 더해주었다 .
