# 프론트엔드 웹 개발의 3대 요소

웹 페이지를 만드는 가장 기본적이면서 중요한 세 가지 요소인 HTML, CSS, JavaScript에 대해 알아봅니다.

## 1. HTML (Hyper Text Markup Language)
**HTML은 웹 페이지의 '뼈대'를 담당합니다.**

- **역할**: 웹 페이지의 구조를 정의하고 콘텐츠(텍스트, 이미지, 비디오 등)를 배치합니다.
- **특징**: 태그(`<div>`, `<a>`, `<h1>` 등)를 사용하여 요소의 의미를 브라우저에게 전달합니다.
- **비유**: 건물을 지을 때 설계도면이나 철골 구조와 같습니다.

## 2. CSS (Cascading Style Sheets)
**CSS는 웹 페이지의 '옷'을 담당합니다.**

- **역할**: HTML로 만든 뼈대에 디자인을 입힙니다. 색상, 폰트, 간격, 레이아웃 등을 조절합니다.
- **특징**: 선택자(Selector)를 사용하여 스타일을 적용하며, 반응형 디자인(Media Query)을 통해 다양한 기기에 최적화된 화면을 제공합니다.
- **비유**: 건물의 외벽 페인트칠, 인테리어 소품과 같습니다.

## 3. JavaScript
**JavaScript는 웹 페이지의 '근육'과 '뇌'를 담당합니다.**

- **역할**: 웹 페이지에 생명력을 불어넣어 동적인 움직임을 구현합니다. 사용자와의 상호작용(클릭, 스크롤, 데이터 입력 등)을 처리합니다.
- **특징**: 브라우저에서 실행되는 유일한 프로그래밍 언어로, 현대 웹 개발에서는 React, Next.js 등 라이브러리와 프레임워크의 기반이 됩니다.
- **비유**: 건물의 엘리베이터 작동, 자동문 시스템과 같습니다.

---

### 결론
웹 개발은 이 세 가지 요소가 유기적으로 결합되어 완성됩니다. HTML로 의미 있는 구조를 잡고, CSS로 아름답게 꾸미며, JavaScript로 편리한 기능을 더함으로써 사용자에게 최상의 경험을 제공할 수 있습니다.
