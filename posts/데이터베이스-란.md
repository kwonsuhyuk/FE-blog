# 데이터 베이스란?
데이터를 통합하여 효율적으로 관리하기 위한 데이터 집합체.
데이터 구조화 하여 관리 
- 데이터 중복 막음
- 효율적, 빠른 데이터 연산
---

## DBMS
(DataBase Management System)
- 데이터베이스를 운영 관리 위해 사용


---

## SQL 
(structured query language)

> Docker통해 mariadb 사용
```bash
docker run -p 3306:3306 --name {컨테이너 이름 작성} -e MARIADB_ROOT_PASSWORD={비밀번호} -d mariadb
// mariadb가 있는 컨테이너 접속 
docker exec  -it mariadb /bin/bash
//mariadb 실행 
mariadb -u root -p
```

> 테이블 생성
```sql 
CREATE DATABASE Tennis;
USE Tennis;
CREATE TABLE member
    -> (
    ->      id VARCHAR(30),
    ->      name VARCHAR(30),
    ->      pwd VARCHAR(30)
    -> );
```

>테이블 데이터 조회
- SELECT 컬럼명 FROM 테이블명;

> 특정 데이터 조회
- SELECT 컬럼명 FROM 테이블명 WHERE 조건

>테이블 데이터 삽입
- INSERT 컬럼명1, 컬럼명2, ... INTO 테이블명
  VALUES (컬럼1 데이터, 컬럼2 데이터, ...);

>테이블 데이터 수정
- UPDATE 테이블명 SET 컬럼명 = 수정할 값 WHERE 조건;

> 테이블 데이터 삭제
- DELETE FROM 테이블명 WHERE 조건;

## 데이터베이스와 nodejs 연결하기

1. my sql 프로젝트에 설치
```bash
npm i mysql
```

2. **_database/connect/_** 경로에 **mariadb.js** 생성

```js
const mariadb = require("mysql");

const conn = mariadb.createConnection({
  host: "localhost",
  port:   ,// 포트번호
  user: "root", //유저 이름 
  password: "root", // db pw
  database: "Tennis", // database 이름
});

module.exports = conn;
```

3. index.js 에 connect로 연결
```js
const mariadb = require("./database/connect/mariadb");
mariadb.connect();
```
